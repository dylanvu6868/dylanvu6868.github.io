// 03 — Selected work

Flagship · CV + LLM RAG

AgriRAG — Rice Leaf Disease Recognition with RAG

An end-to-end system that diagnoses 10 rice-leaf diseases from a photo, then explains the treatment with citations. EfficientNetB0 with two-stage transfer learning hits 99.01% test accuracy across 15,023 images, with Focal Loss and inverse-frequency weighting to handle a 1.8:1 imbalance.

The grounded-answer layer is a hybrid RAG pipeline — FAISS dense retrieval, BM25 keyword search, reciprocal-rank fusion and cross-encoder re-ranking — with smart, structure-aware chunking and parent–child retrieval over a ~340-page agricultural corpus, served by a FastAPI + SSE backend on LLaMA-3.3-70B.

Python · EfficientNetB0 · MobileNetV3 · FAISS · BM25 · RRF · LLaMA 3.3 · FastAPI
